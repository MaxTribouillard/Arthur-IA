# Arthur — Employé IA & Rédacteur de Contenu Autonome
<img width="687" height="768" alt="arthur" src="https://github.com/user-attachments/assets/9ad5e3cd-50b3-4e07-8b47-6dc002ee3336" />


Arthur est un employé IA capable de gérer l'intégralité du cycle de vie d'un blog technique de manière 100% autonome. Grâce à un pipeline d'agents spécialisés et un workflow structuré, il réalise la veille sectorielle, rédige des articles optimisés pour le SEO au format Markdown, met à jour l'index du site et déploie automatiquement les fichiers sur le serveur de production via un outil personnalisé (Custom Tool) SFTP.
